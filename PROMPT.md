# Промт для AI Studio: Главная страница Indigo Lab

## Задача
Создай главную страницу для сайта **indigolab.pro** — лендинг-хаб проекта Indigo Lab Андрея Индиго.

## Контекст
Indigo Lab — это экосистема продуктов вокруг авторского метода **БПСП** (биопсихосоциальное программирование):
- Обучение психологов и коучей методу БПСП
- Работа с клиентами по устранению РПП (расстройств пищевого поведения)
- B2B-программа для бизнеса
- Научные исследования и публикации

Сейчас есть два старых сайта на Tilda (их контент нужно забрать) и два новых SPA в правильном стиле (b2b.indigolab.pro и certified.indigolab.pro — их дизайн нужно взять за основу).

## Референсы дизайна (ЭТАЛОН)
Смотри на **b2b.indigolab.pro** и **certified.indigolab.pro** — это правильный стиль:
- Тёмная тема (фон: `#13102a`, глубокий фиолетовый)
- Стеклянные карточки (glassmorphism)
- Градиентные CTA-кнопки (violet → magenta → розовый)
- Nebula-градиент на фоне (тройной radial-gradient)
- Типографика: Manrope (основной) + Fraunces (декоративный)
- Много воздуха, крупные заголовки, плавные анимации

**Дизайн-токены**: см. файл `design/tokens.css`

## Источники контента
В папке `content/from-tilda/` лежат тексты из старых Tilda-сайтов:
- `indigo-lab-home.txt` — старая главная IndigoLab (РПП-фокус)
- `indigo-people-start.txt` — старая главная Indigo People (визитка Андрея)
- `indigo-people-about.txt` — страница «Об Андрее»
- `indigo-people-metod.txt` — страница о методе БПСП

**Контент-бриф** с точной структурой: `content/main-page.md`

## Технические требования
1. **React 18+** (Vite), **Tailwind CSS 4**
2. **shadcn/ui** компоненты где уместно
3. Использовать CSS-переменные из `design/tokens.css`
4. Все изображения, иконки — локально в `/assets/`
5. **Адаптив**: mobile-first, от 375px до 1920px
6. **Тёмная тема** — единственная, без переключения
7. **OG-метатеги** для соцсетей
8. **Фавиконка**
