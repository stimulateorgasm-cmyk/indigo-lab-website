# Референс: b2b.indigolab.pro

## URL
https://b2b.indigolab.pro

## Описание
B2B-лендинг Indigo Lab. Программа для бизнеса: «Превращаем внутреннее напряжение команды в измеримый рост прибыли».

## Характеристики
- **Стек**: React 18 + Vite + Tailwind CSS 4
- **Тема**: Dark-only (`<html class="dark">`)
- **Цвет фона**: `#13102a`
- **Шрифты**: Google Fonts (предположительно Manrope)
- **Метатеги**: OG + Twitter Card
- **Фавикон**: SVG

## Визуальные особенности
- Hero с nebula-градиентом
- Стеклянные карточки с низкой прозрачностью
- Градиентные CTA-кнопки
- Анимации fade-in + slide-up
- Акцентные линии с градиентом
- Теневые эффекты (halo, glow)

## Структура страницы
1. Hero с УТП
2. Проблематика (боли бизнеса)
3. Решение (программа INDIGO)
4. Как это работает
5. Кейсы / результаты
6. Отзывы
7. Команда
8. FAQ
9. Форма заявки
10. Футер

## Ключевые дизайн-решения
- Трёхцветный CTA-градиент: violet → magenta → розовый → violet
- Nebula-фон: тройной radial-gradient с oklab-цветами
- Glass-эффект: `inset 0 1px 0 oklab(100% 0 0 / .1)`
- Свечение кнопок: `0 10px 40px -10px oklab(72% .225 -.13 / .6)`
