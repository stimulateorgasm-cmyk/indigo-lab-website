# Референс: certified.indigolab.pro

## URL
https://certified.indigolab.pro

## Описание
Страница сертифицированных специалистов БПСП. Карточки специалистов с фото.

## Характеристики
- **Стек**: React 18 + Vite + Tailwind CSS 4
- **Тема**: Dark-only
- **Цвет фона**: тёмный (тот же диапазон что b2b)

## Отличия от b2b
- Более светлый акцент (зелёный `#00bb7f` для статуса «сертифицирован»)
- Карточки специалистов с фото
- Статус-бейджи
