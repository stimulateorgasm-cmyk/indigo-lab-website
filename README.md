# Indigo Lab — Новый сайт

Пересборка indigolab.pro: объединение двух Tilda-проектов в современный SPA.

## Что здесь
```
indigo-lab-website/
├── README.md                 ← этот файл
├── PROMPT.md                 ← промт для AI Studio
├── design/
│   ├── README.md             ← описание дизайн-системы
│   └── tokens.css            ← CSS-переменные (цвета, градиенты, тени)
├── content/
│   ├── main-page.md          ← контент-бриф главной страницы
│   └── from-tilda/           ← контент из старых Tilda-сайтов
│       ├── indigo-lab-home.txt
│       ├── indigo-people-start.txt
│       ├── indigo-people-about.txt
│       └── indigo-people-metod.txt
└── references/
    ├── b2b.md                ← описание b2b.indigolab.pro
    └── certified.md          ← описание certified.indigolab.pro
```

## Эталонный дизайн
- [b2b.indigolab.pro](https://b2b.indigolab.pro) — B2B-лендинг (тёмная тема, стекло, градиенты)
- [certified.indigolab.pro](https://certified.indigolab.pro) — Специалисты (тот же стиль)

## Стек
React 18, Vite, Tailwind CSS 4, shadcn/ui

## Этапы
1. 🚧 Главная страница (AI Studio) ← сейчас здесь

4. ⬜ Страница «РПП» (из Tilda)
